program author: Yuelin Liu
purpose: this program simulates a running match between a tortoise and a hare
source file: graphics.c, init.c, main.c, thread.c
header file: defs.h

compilation and launching instruction:
1. download tar file, open terminal and move the directory to the tar file
2. unzip tar file by tar -xvf a4.tar
3. compile file by make
4. execute file by ./a4
